# 🧬 Idea Survival Simulator

An AI-based system that analyzes startup ideas and predicts their success probability.

## 🚀 Features
- Market Demand Analysis
- Competition Detection
- Risk Evaluation
- Survival Score Prediction
- Improvement Suggestions

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- AI Logic (Kiro Generated)

## 💡 How it Works
User inputs a startup idea → System analyzes multiple factors → Displays survival probability and suggestions.

## 📌 Future Scope
- Add real-time market data
- Integrate ML models
- Advanced analytics dashboard
